# MapleEngine
DirectX12　描画エンジン
## 2D
-  スプライト描画
-  一部切り抜き描画
## 3D
- .Obj並びに.gltf読み込み
- モデル描画
- animation再生
- 2BoneIK(現在制作中)
- Phong shading
- ポストエフェクト
## その他
上記以外にも各種便利関数や、数学関数が存在しています。
詳細は各ヘッダファイルを参照してください。
## 動作確認済み環境
Windows10 64bit
